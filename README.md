# Portfolio Atlas

## Project Overview
A Django based portfolio to get a easy feel for the framework. Making 
use of the opportunity to both create a portfolio and add a project to
the portfolio. 

I enjoy history inspired project names, Atlas for my portfolio as it 
will carry the other projects on its shoulders.
